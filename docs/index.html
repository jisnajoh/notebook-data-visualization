Build Steps

Click a step to see the implementation notes.

01 Initialize Next.js 16 + install dependencies
create-next-app with TypeScript, Tailwind 4, App Router, and Turbopack. Install chart.js, react-chartjs-2, chartjs-chart-matrix, papaparse, firebase, react-syntax-highlighter, and @dnd-kit.
02 Firebase + layout
lib/firebase.ts initialises Firebase app with env vars. Root layout updates metadata title and global CSS sets the dark colour palette and scrollbar styles.
03 Core lib files
chartDataProcessor.ts: detectColumnType (with numeric-string exclusion for dates), cleanCSV, processCSV, buildChartData. Supporting libs: chartTypes, chartConfig, advancedDataProcessors (KDE, histogram, skewness), visualizationAnalysis (COLUMN_KNOWLEDGE engine), visualizationStorage.
04 Pyodide hook + chart components
usePyodide.ts loads Pyodide from CDN and pre-installs pandas/numpy/matplotlib/scipy. All 10 Universal chart components built with Chart.js. PairPlot uses responsive CSS Grid with aspect-ratio cells scaling maxHeight by column count.
05 CsvVisualizations + NotebookViewer
CsvVisualizations offers STANDARD (sidebar nav) and CUSTOM (gallery picker) modes. Each chart panel has type-filtered column selectors. NotebookViewer renders 14 CodeCells; matplotlib output is captured as __IMG_BASE64__ and rendered inline.
06 CsvNotebookBuilder orchestrator
5-tab workspace: Upload (drag-drop + samples), Cleaning (type table + row preview), Notebook, Visualizations, Account. Firebase auth + Firestore cloud save, local-storage fallback. Delete with confirmation dialog.
07 GitHub Pages portal
Vanilla HTML/CSS/JS docs site with animated KPI counters, interactive step explorer, and Chart.js evaluation charts switchable between Academic / SME / Enterprise profiles.
08 Final wiring + build verification
npm run build to catch TypeScript and ESLint errors. Fix any type assertions, missing imports, or unused variables. Confirm npm run dev starts cleanly and the upload → clean → notebook → visualize flow works end-to-end.
